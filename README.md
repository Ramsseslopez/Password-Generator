# Password-Generator

![image](https://user-images.githubusercontent.com/112135143/193921037-164c05bd-60a2-4a13-ac8c-aab6edcbcd19.png)


## Description
Developed using HTML, CSS, and Javascript. The Password-Generator was built to generate random passwords using a length of 8 and 128 characters.
 
 ## Usage
 The Password-Generator consists of Lower caser, Upper case, Numbers, and special characters. Feel free to generate your own Password!.

Link to deployed page: https://ramsseslopez.github.io/Password-Generator/
